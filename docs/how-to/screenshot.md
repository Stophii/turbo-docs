# Take a Screenshot

On native platforms, you can capture a screenshot from your game via keyboard shortcut:

<Tabs>
    <TabItem value="macos_linux" label="MacOS / Linux" default>
    `Cmd + 1`
    </TabItem>
    <TabItem value="windows" label="Windows" default>
    `Ctrl + 1`
    </TabItem>
</Tabs>
