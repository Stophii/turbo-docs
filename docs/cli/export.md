# `turbo export`

```md
Exports a Rust Turbo project in web format

Usage: turbo export [OPTIONS] [PROJECT_DIR]

Arguments:
  [PROJECT_DIR]  [default: .]

Options:
  -c, --config <CONFIG>          [default: ]
  -o, --export-dir <EXPORT_DIR>
  -b, --include-virtual-gamepad
  -h, --help                     Print help
  -V, --version                  Print version
```
