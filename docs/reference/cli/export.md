# `turbo export`

```md
Exports a Rust Turbo project in web format

Usage: turbo export [OPTIONS] <PROJECT_DIR>

Arguments:
  <PROJECT_DIR>  

Options:
  -n, --app-name <APP_NAME>                  [default: "My Turbo Game"]
  -v, --app-version <APP_VERSION>            [default: 0.0.0]
  -a, --app-author <APP_AUTHOR>              [default: "Leeroy Jenkins"]
  -d, --app-description <APP_DESCRIPTION>    [default: "An awesome game about awesomeness"]
  -x, --app-resolution-x <APP_RESOLUTION_X>  [default: 256]
  -y, --app-resolution-y <APP_RESOLUTION_Y>  [default: 144]
      --http-rpc-url <HTTP_RPC_URL>          [default: http://localhost:8899]
      --ws-rpc-url <WS_RPC_URL>              [default: ws://localhost:8900]
  -h, --help                                 Print help
  -V, --version                              Print version
```
