# `turbo export`

```md
Exports a Rust Turbo project in web format

Usage: turbo export [OPTIONS] [PROJECT_DIR]

Arguments:
  [PROJECT_DIR]  [default: .]

Options:
  -n, --app-name <APP_NAME>                  
  -v, --app-version <APP_VERSION>            
  -a, --app-author <APP_AUTHOR>              
  -d, --app-description <APP_DESCRIPTION>    
  -x, --app-resolution-x <APP_RESOLUTION_X>  [default: 256]
  -y, --app-resolution-y <APP_RESOLUTION_Y>  [default: 144]
      --http-rpc-url <HTTP_RPC_URL>          [default: ]
      --ws-rpc-url <WS_RPC_URL>              [default: ]
  -o, --export-dir <EXPORT_DIR>              
  -h, --help                                 Print help
  -V, --version                              Print version
```
